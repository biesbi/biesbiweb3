   - rootMargin: 50px (çok agresif)
   - Placeholder: Gri arka plan (#f5f5f5)

  ⚙️ TEKNİK DETAYLAR
================================================================================

LAZY LOADING NASIL ÇALIŞIR?

1. Sayfa yüklendiğinde:
   - Tüm <img> etiketlerinin SRC'si KALDIRILIR
   - data-original-src attribute'una kaydedilir
   - Placeholder (gri arka plan) gösterilir

2. Intersection Observer:
   - Her resmi izler
   - Ekranda görünür olunca (50px önceden)
   - SRC'yi geri yükler
   - Resim yüklenmeye başlar

3. Sonuç:
   - İlk yüklemede 0 resim yüklenir
   - Sadece görünür olanlar yüklenir
   - Çok daha hızlı sayfa açılışı


PAGINATION NASIL ÇALIŞIR?

1. API Request:
   GET /api/products?limit=20&page=1

2. API Response Headers:
   X-Has-More: 1  (daha fazla var)
   X-Page: 1
   X-Limit: 20

3. Frontend:
   - Response'u alır
   - 20 ürün gösterir
   - X-Has-More=1 ise "Daha Fazla" butonu gösterir
   - Butona tıklanınca page++ yapıp tekrar request atar
