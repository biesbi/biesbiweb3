🐛 Gallery Debug Tool

⚠️ Bu sayfa admin paneldeki fotoğraf yükleme akışını debug etmek için hazırlandı.

📋 Bilgilendirme

Admin panelde fotoğraf yükleme şu şekilde çalışıyor:

  1. Fotoğrafları seçiyorsunuz → pendingUploads kuyruğuna ekleniyor
  2. Ürünü kaydediyorsunuz (Kaydet butonuna basıyorsunuz)
  3. Ürün başarıyla kaydedilince → Kuyruktaki fotoğraflar otomatik yükleniyor

Önemli: Mevcut bir ürüne fotoğraf eklemek için de ürünü tekrar kaydetmeniz gerekiyor!
